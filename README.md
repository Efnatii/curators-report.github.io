# Curators Report

Простой статический сайт для размещения подборок материалов, заметок и полезных ссылок. Проект рассчитан на публикацию через GitHub Pages.

## Структура
- `index.html` — разметка главной страницы.
- `styles.css` — базовые стили оформления.

## Локальный просмотр
Откройте `index.html` в браузере или поднимите простой веб-сервер, например:

```sh
python -m http.server 4000
```

После этого сайт будет доступен по адресу <http://localhost:4000>.
