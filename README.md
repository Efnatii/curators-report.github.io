# Curators Report

Простой статический сайт-анкета для сбора и экспорта отчетов кураторов. Работа происходит полностью в браузере — без серверной части и регистрации пользователей, поэтому страницу можно открыть локально или опубликовать через GitHub Pages.

## Описание сайта
- **Форма-анкета для кураторов.** Пять тематических разделов с текстовыми полями, выпадающими списками и датами, которые помогают зафиксировать общий профиль куратора, сведения о группе, успеваемость, внеучебную деятельность и общий вывод.
- **Динамическое добавление блоков.** У полей, которые могут повторяться (должности, состав группы, мероприятия), есть кнопки «Добавить», чтобы фиксировать несколько значений.
- **Отчетный период.** В шапке можно выбрать период из списка, который автоматически подставляет годы учебных семестров.
- **Сохранение в браузере.** Заполненная форма хранится в `localStorage`, чтобы не терять данные при обновлении страницы.
- **Экспорт для дальнейшей обработки.** Кнопка «Скачать ответы» создает JSON со значениями полей и подсказками.

## Что внутри
- `index.html` — разметка главной страницы анкеты с подсказками, выпадающими списками и динамическими блоками для добавления нескольких значений.
- `styles.css` — базовые стили оформления и адаптивная сетка.
- `combine_json_to_excel.py` — скрипт для объединения экспортированных JSON-файлов в один Excel-файл.

## Локальный просмотр
Откройте `index.html` в браузере или поднимите простой веб-сервер, например:

```sh
python -m http.server 4000
```

После этого сайт будет доступен по адресу <http://localhost:4000>. Форма хранится только локально в браузере и не требует бекенда.

## Экспорт ответов
- Кнопка «Скачать ответы» формирует JSON-файл с полями анкеты и параллельно сохраняет PDF-версию заполненной формы.
- В объект добавлен блок `_comments` с пояснениями к каждому полю, чтобы данные можно было быстрее понять при последующей обработке.
- Повторяющиеся поля (например, должности, номера групп или перечень мероприятий) сохраняются как массивы, поэтому удобно анализируются при дальнейшей агрегации.

## Объединение ответов в Excel
1. Соберите все экспортированные JSON-файлы в одну папку.
2. Запустите скрипт объединения:
   ```sh
   python combine_json_to_excel.py path/to/json_dir combined.xlsx
   ```
3. На выходе появится Excel-файл с колонками в порядке вопросов анкеты. Списки значений подсвечиваются фоном, чтобы было видно блок строк, относящийся к одному ответу.

## Требования
- Python 3.10+ для работы со скриптом объединения. При первом запуске автоматически установится `openpyxl`.
- Любой современный браузер для заполнения и выгрузки анкеты.
